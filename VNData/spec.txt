- SCRIPT [p1: String]
"p1.vns" 파일을 부르고 실행합니다.


- SET [p1: Symbol], [p2: any]
p1 변수에 p2 값을 넣습니다.
모든 타입을 지정할 수 있습니다.


- ADD [p1: Symbol], [p2: Number]
p1 변수에 p2 값을 더합니다.
p1 변수는 Number 타입이여야합니다.


- SUB [p1: Symbol], [p2: Number]
p1 변수에 p2 값을 뺍니다.
p1 변수는 Number 타입이여야합니다.


- MUL [p1: Symbol], [p2: Number]
p1 변수에 p2 값을 곱합니다.
p1 변수는 Number 타입이여야합니다.


- DIV [p1: Symbol], [p2: Number]
p1 변수에 p2 값을 나눕니다.
p1 변수는 Number 타입이여야합니다.


- MOD [p1: Symbol], [p2: Number]
p1 변수에 p2 값을 나눴을 때의 나머지를 입력합니다.
p1 변수는 Number 타입이여야합니다.


- FLOOR [p1: Symbol]
p1 변수의 소수점 아래의 수치를 버립니다.
p1 변수는 Number 타입이어야합니다.


- ROUND [p1: Symbol]
p1 변수의 소수점 아래의 수치를 반올림하여 버리거나 올립니다.
p1 변수는 Number 타입이어야합니다.


- CEIL [p1: Symbol]
p1 변수의 소수점 아래의 수치를 올립니다.
p1 변수는 Number 타입이어야합니다.


- STRING [p1: Symbol]
p1 변수의 값을 String 타입으로 변경합니다.
각 타입에 맞는 문자열로 변경됩니다.


- CONCAT [p1: Symbol], [p2: String]
p1 변수의 값에 p2 문자열을 잇습니다.
p1 변수는 String 타입일 필요는 없습니다.


- IF [p1: any], [p2: Symbol], [p3: any]
- END
p1 값과 p3 값을 p2 연산자로 비교합니다.
p2 값은 ==, !=, >, <, >=, <= 중 하나여야합니다.
조건에 일치하는 경우 다음 줄의 명령을 실행합니다.
조건에 일치하지 않는 경우, 같은 깊이의 END까지 건너뜁니다.


- LABEL [p1: Symbol]
- GOTO [p1: Symbol]
p1로 정의된 LABEL 명령이 위치한 코드로 실행 위치를 변경합니다.


- UNLOCK [p1: Symbol], [p2: String]
p1 값에 따라 결정되는 내용을 해금합니다.
p1 값은 대소문자를 구별합니다.
  * p1 name
    대사에 표시되는 이름이 ???로 표시되지 않도록 설정합니다.


- TEXT [p1: String]
대사창에 화자가 없는 텍스트를 표시합니다.


- SAY [p1: String], [p2: String]
대사창에 p1을 화자로 하는 텍스트를 표시합니다.
UNLOCK name 명령으로 화자가 해금되지 않았다면 ???로 표시됩니다.


- SEL [p1: String], ...
선택지를 표시합니다.
인자 수는 제한되지 않으며, 모든 인자가 각 선택지가 됩니다.
$SEL 변수에 선택한 선택지의 번호가 입력됩니다.
선택지 번호는 1부터 시작합니다.


- BGM [p1: String]
"BGM/p1.mp3" 음악을 반복 재생합니다.
이미 재생중인 음악이 있다면 중단하고 재생합니다.

- BGM [p1: Null]
배경 음악을 중단합니다.


- BG [p1: String]
"BG/p1.png" 이미지를 배경으로 불러옵니다.
이미 표시중인 배경 이미지가 있다면 변경됩니다.

- BG [p1: Null]
배경 이미지를 없앱니다.


- SCG [p1: Number], [p2: String], [p3: Position]
"SCG/p2.png" 이미지를 p1번에 불러옵니다.
이미 p1번에 이미지가 존재했다면 해당 이미지를 없앤 후에 불러옵니다.
불러온 이미지는 p3 위치에 표현됩니다.
$ p1은 정수형이어야 합니다.
$ p3은 left, center, right 중 하나여야합니다.

- SCG [p1: Number], [p2: Null]
p1번 이미지를 없앱니다.


- FX [p1: Symbol]
p1에 해당하는 효과를 실행합니다.


- FREEZE
현재 화면이 갱신되지 않도록 고정합니다.
화면이 이미 고정되었다면 오류가 발생합니다.


- TRANSITION [p1: Number]
고정된 화면에서 현재 그려질 화면으로 p1초동안 서서히 변경됩니다.

- TRANSITION [p1: Number], [p2: String]
고정된 화면에서 현재 그려질 화면으로 p1초동안 서서히 변경됩니다.
전환 효과는 "Transition/p2.png" 이미지를 참조합니다.
검정색에서 하얀색으로 진행될수록 결과 화면에 가깝게 진행됩니다.
